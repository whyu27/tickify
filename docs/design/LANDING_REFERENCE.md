# 11. AI Implementation Rules

Saat mengimplementasikan Landing Page:

- Gunakan UI_GUIDELINES.md sebagai acuan utama.
- Pertahankan hierarchy setiap section.
- Jangan mengubah urutan section.
- Gunakan reusable components.
- Jangan menambahkan section tanpa persetujuan.
- Layout harus konsisten dengan referensi visual.
- Landing Page menjadi dasar desain seluruh aplikasi Tickify.
